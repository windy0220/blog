写作
```bash
hexo new post_name #新建博文
hexo new page page_name #新建页面
```

编译部署
```bash
hexo clean #清空public目录
hexo generate #生产静态文件
hexo server #创建本地服务器 
hexo deploy #部署网站
```

配置新的环境
```bash
git clone git@git.oschina.net:windy0220/bigma.cc.git
npm install hexo
npm install
npm install hexo-deployer-git
# 注意不需要hexo init
```
